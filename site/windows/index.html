Known Issues

How to fix MBR

Fix MBR with command prompt and installation DVD. Fix MBR with command prompt requires the original Windows 8 or 8.1 installation DVD. The instructions are

1.Boot from the original installation DVD. 2.At the Welcome screen, click "Repair your computer". 3.Choose" Troubleshoot > Advanced options > Command Prompt". 4.When the Command Prompt loads, type the following commands in order:

bootrec /FixMbr
bootrec /FixBoot
bootrec /ScanOs
bootrec /RebuildBcd

Remember to press "Enter" after each command and wait for each operation to finish.

5.Remove the DVD from the disk tray. 6.Type exit and Hit "Enter". 7.Restart your computer and check if Windows 8/8.1 can boot now.
